ENI - Memory Card Game

Welcome on this Memory Card Game ! Enjoy the game and do your best !

Règles du jeu:

- Retournez deux cartes. Si les images sont identiques, le jeu les gardera face visible.
- Si les images sont différentes elles se retournent faces cachées là où elles étaient.
- La partie est terminée quand toutes les cartes sont retournées en paires.

Si vous voulez vous souvenir de votre score, inscrivez vous.
- Le jeu garde les 5 meilleurs scores.
- Si vous voulez relancer une partie cliquez sur la barre espace.
